<details>
<summary>修改 App 在手机上展示的名称</summary>

### Android

修改 `android/app/src/main/res/values/strings.xml` 配置

```xml
<resources>
  <string name="app_name">这里填写名称</string>
</resources>
```

### iOS

修改 `ios/AneConfigure/Info.plist` 配置

```xml
	<key>CFBundleDisplayName</key>
	<string>这里填写名称</string>
```
